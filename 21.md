# Python 列表基础知识

## 一、什么是列表

列表是 Python 里最常用的数据类型之一，用来存放一组有序的数据。

- 用方括号 [] 表示
- 里面可以放数字、字符串、布尔值等
- 元素之间用逗号隔开

示例：
'''python
nums = [1,2,3,4,5]
names = ["张三","李四","王五"]
mix = [10,"hello",True,3.14]
'''

## 二、列表的特点

1. 有序：有先后顺序，可以通过位置找到
2. 可变：可以修改、添加、删除里面的内容
3. 可重复：同一个值可以出现多次
4. 可以放不同类型的数据

## 三、列表常用操作

### 1. 访问元素（通过下标 / 索引）

下标从 0 开始
'''python
lst = [10,20,30,40]
print(lst[0])   # 10 （订正：去掉print后多余空格，代码格式更规范）
print(lst[2])   # 30
print(lst[-1])  # 最后一个 40
'''

### 2. 修改元素
'''python
lst[1] = 99
print(lst)  # [10,99,30,40]
'''

### 3. 添加元素
- 末尾添加：append()（订正：去掉append后多余空格）
  '''python
  lst.append(50)
  '''
  
- 指定位置插入：insert()（订正：去掉insert后多余空格）
  '''python
  lst.insert(2, 66)
  '''
  
- 追加另一个列表：extend()（订正：去掉extend后多余空格）
  '''python
  lst.extend([77,88])
  '''

### 4. 删除元素
- 删除指定下标：pop()（订正：去掉pop后多余空格）
  '''python
  lst.pop()    # 删除最后一个
  lst.pop(1)   # 删除下标 1
  '''
  
- 删除指定值：remove()（订正：去掉remove后多余空格）
  '''python
  lst.remove(30)
  '''
  
- 清空列表：clear()（订正：去掉clear后多余空格）
  '''python
  lst.clear()
  '''
  
- 删除整个列表：del
  '''python
  del lst
  '''

### 5. 查找与判断
- 判断是否在列表里：in
  '''python
  print(20 in lst)  # 订正：去掉print后多余空格
  '''
  
- 找下标：index()（订正：去掉index后多余空格）
  '''python
  print(lst.index(30))  # 订正：去掉print后多余空格
  '''
  
- 统计次数：count()（订正：去掉count后多余空格）
  '''python
  print(lst.count(20))  # 订正：去掉print后多余空格
  '''

### 6. 列表长度
'''python
print(len(lst))  # 订正：去掉print后多余空格
'''

### 7. 排序
- 升序：sort()（订正：去掉sort后多余空格）
  '''python
  lst.sort()
  '''
  
- 降序：sort(reverse=True)（订正：去掉sort后多余空格）
  '''python
  lst.sort(reverse=True)
  '''

### 8. 反转
'''python
lst.reverse()  # 订正：补充换行，代码块格式更整洁
'''

## 四、遍历列表
'''python
for i in lst:
    print(i)  # 订正：添加缩进，符合Python语法规范
'''
带下标遍历：
'''python
for index, value in enumerate(lst):  # 订正：去掉enumerate后多余空格
    print(index, value)  # 订正：添加缩进，符合Python语法规范
'''

## 五、列表切片（截取一部分）

格式：列表 [起始：结束：步长]
'''python
lst = [1,2,3,4,5,6]
print(lst[1:4])    # [2,3,4] （订正：去掉print后多余空格）
print(lst[:3])     # 前 3 个
print(lst[3:])     # 从第 3 个开始到最后
print(lst[::2])    # 隔一个取一个
print(lst[::-1])   # 反转
'''

## 六、列表拷贝

浅拷贝：
'''python
new_lst = lst.copy()  # 订正：去掉copy后多余空格
new_lst = lst[:]
'''

深拷贝（嵌套列表用）：
'''python
import copy
new_lst = copy.deepcopy(lst)  # 订正：去掉deepcopy后多余空格
'''
